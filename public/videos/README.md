# Hero video

Drop a file named `amt-hero.mp4` here to enable the hero video background.

- Resolution: 1920x1080 minimum
- Codec: H.264 (mp4)
- Length: 15–25 seconds, seamlessly looping
- Audio: removed (the player is muted by design)
- Weight: aim for < 4 MB (the page falls back to a 3D scene if the video is missing or fails to load)
